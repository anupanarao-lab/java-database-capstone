# User Stories for Healthcare System

## 1. Doctor

### As a Doctor, I want to:
- **View my appointments** so that I can plan my day efficiently.
- **Update patient medical records** so that the treatment history is accurate.
- **Prescribe medications** so that patients receive proper treatment.
- **Receive notifications** about urgent patient requests so that I can respond quickly.
- **Access patient history** so that I can make informed medical decisions.

---

## 2. Patient

### As a Patient, I want to:
- **Register and login** so that I can access my personal health information.
- **Book appointments** so that I can consult a doctor at a convenient time.
- **View my medical history** so that I can track my health records.
- **Receive reminders** for upcoming appointments so that I do not miss them.
- **Request prescription refills** so that I can continue my medications without interruptions.

---

## 3. Admin

### As an Admin, I want to:
- **Manage user accounts** (Doctors and Patients) so that only authorized users access the system.
- **View system analytics** so that I can monitor system usage.
- **Handle appointment approvals** so that scheduling conflicts are minimized.
- **Send system notifications** so that important updates reach users on time.
- **Generate reports** on patients, appointments, and doctors so that the system performance can be reviewed.
